# QS University Ranking Analysis

This project investigates the QS university ranking methodology by testing whether the published factor weights (employability, research output, studentâ€“faculty ratio, international students, etc.) align with real-world data. Using a linear regression model on available university data, the project estimates factor contributions and compares them to the official QS weights, while noting limitations due to missing factors in the dataset.

## Methodology

The official QS ranking methodology can be found here: [QS Ranking Methodology](https://www.topuniversities.com/world-university-rankings/methodology)

## Dataset

The dataset used in this project was taken from: [Global University Rankings 2025-2026 on Kaggle](https://www.kaggle.com/datasets/zulqarnain11/global-university-rankings-2025-2026/data)
